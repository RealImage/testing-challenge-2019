1- This code snippet has the automation test code for the Sharebox APIs.
2- It has 6 test cases as below.
	2.1 List all the files
	2.2 Upload file
	2.3 Check uploaded file
	2.4 Delete file
	2.5 Share file to others.
	2.6 Approve/Reject file from other user.
3- It has a file called runner.py , which launches the code and it start with tests.
4- Folder sharebox has 3 files, upload.py, files.py and common.py.
5- runner.py calls for upload.py for upload APIs and files.py for files API.
6- common.py is called from files.py and upload.py.
